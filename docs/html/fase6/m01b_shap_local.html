SHAP Local — Explicaciones Individuales

Explicaciones a nivel de alumno individual con SHAP sobre CatBoost. Se analizan tres perfiles representativos: un alumno que abandonó (verdadero positivo), uno que no abandonó (verdadero negativo) y uno en zona de máxima incertidumbre (probabilidad predicha ~0.5). Para cada perfil se muestra cómo cada variable empuja la predicción hacia abandono o hacia permanencia.

Waterfall — Alumno que no abandona (verdadero negativo)

Las barras azules dominan: el modelo detecta factores protectores que reducen el riesgo de abandono por debajo del umbral de decisión.

Waterfall — Alumno en zona de riesgo (probabilidad ~0.5)

Contribuciones mixtas: factores de riesgo y protectores se compensan. Este perfil es el más difícil de clasificar y el más relevante para intervención preventiva.

Decision plot — Comparativa de los 3 perfiles

Cada línea muestra cómo se acumula el valor SHAP feature a feature. La separación entre líneas indica qué variables distinguen más a los tres perfiles.

Uso práctico: Las explicaciones locales permiten a los orientadores universitarios entender por qué el modelo alerta sobre un alumno concreto, más allá del ranking global de features. Cada alumno tiene su propio perfil de riesgo.
