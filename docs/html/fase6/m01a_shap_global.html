Fase 6 — SHAP Global: Importancia de Features

Análisis de importancia global con SHAP sobre el conjunto de test (6.725 observaciones, 27 features). CatBoost y RF mediante TreeExplainer; EBM mediante su método nativo. Incluye dependence plots que revelan umbrales críticos de riesgo.

Beeswarm plot — CatBoost

Cada punto = observación. Eje X = valor SHAP. Color = valor de la feature (rojo=alto, azul=bajo).

Bar plot comparativo — CatBoost · RF · EBM

Importancia media por feature en los tres modelos. Consenso = top en los tres. Nombres legibles en el eje.

Heatmap SHAP — CatBoost (muestra 500 obs)

Mapa de calor de valores SHAP. Patrones de comportamiento del modelo.

🏆 SHAP Interaction Values — Interacciones entre features

Matriz de interacciones entre el top 10 de features. Un valor alto indica que el efecto de una feature sobre el riesgo depende del valor de otra — información que el análisis SHAP estándar no captura. Nivel paper de investigación.

🏆 SHAP Dependence Plots — Umbrales críticos de riesgo (Top 3 features)

Cada punto = alumno. Eje X = valor de la feature, eje Y = contribución SHAP al riesgo. La línea negra es la tendencia media. Revela el umbral exacto a partir del cual cada variable empieza a aumentar el riesgo de abandono. Color = años con beca (interacción socioeconómica): rojo = más años con beca.
